# Borderlands Collection App

A Java-based, menu-driven application built as a final project for my Computer Science coursework.

## Features
- Track collectible items across multiple categories
- Persistent storage using file I/O
- Clean, modular object-oriented design
- Structured user input and validation

## Tech Stack
- Java
- Eclipse
- File I/O (CSV-based)
